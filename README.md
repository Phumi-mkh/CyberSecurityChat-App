## 🎯 Overview

The **Cybersecurity Awareness Chatbot** is an interactive Windows Forms application designed to educate users about online safety. It features a WhatsApp-style chat interface, voice output, memory capabilities, sentiment detection, and keyword recognition for various cybersecurity topics.

### Assignment Part 2 Requirements Met:
- ✅ GUI design with proper spacing and colors
- ✅ ASCII art integration from Part 1
- ✅ Voice implementation
- ✅ Keyword recognition (7 cybersecurity topics)
- ✅ Random responses using Lists/Arrays
- ✅ Conversation flow with follow-up questions
- ✅ Memory feature to recall user information
- ✅ Sentiment detection (worried, frustrated, curious)
- ✅ Error handling for unknown inputs
- ✅ Code optimization with OOP practices

## ✨ Features

### 🤖 Core Functionality
| Feature | Description |
|---------|-------------|
| **Keyword Recognition** | Detects 7 cybersecurity topics: password, phishing, malware, scam, privacy, 2FA, safe browsing |
| **Random Responses** | Multiple responses per topic stored in Lists for varied interactions |
| **Memory Feature** | Remembers user name and personalizes responses |
| **Sentiment Detection** | Responds empathetically to emotions (worried, frustrated, curious, thankful) |
| **Follow-up Questions** | Handles "another tip", "tell me more", "explain more" |

### 🎨 User Interface
- WhatsApp-style chat bubbles
- Color-coded messages (user vs bot)
- Timestamps on every message
- Typing indicator animation
- Quick topic buttons
- Clear chat and clear memory options

### 🔊 Voice Features
- Text-to-speech output using System.Speech
- Toggle voice on/off with button
- Speaks bot responses when enabled

### 💾 Data Management
- Saves chat history to text files
- Memory persists during session
- Clear memory functionality
